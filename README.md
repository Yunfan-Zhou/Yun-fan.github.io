# Yun-Fan Zhou — Academic Website

A lightweight personal academic website focused on radio astronomy, galaxy
cosmology, publications, and research experience.

## Local preview

Run a static server from the repository root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Main files

- `index.html` — page content and metadata
- `assets/css/site.css` — homepage responsive visual design
- `assets/css/cv.css` — formal CV page visual design
- `assets/js/site.js` — navigation and reveal interactions
- `cv.html` — formatted academic CV page
- `images/yunfan-zhou.jpg` — profile portrait
- `files/Yunfan_Zhou_CV.pdf` — original CV source archive, not linked from the primary CV button

The `.nojekyll` file makes GitHub Pages serve the repository as a static site.
